# スポーツ施設自動予約システム開発のプロを募集します

## 概要

【業務内容】
・市営・区営のスポーツ施設の予約サイトにおける空き状況の監視と自動予約システムの構築・運用を担当していただきます。
・指定された条件に基づき、テニスコートの空き状況をリアルタイムで監視し、自動的に予約を行うシステムの開発、改善を行います。
・予約システムの安定稼働と、エラー発生時の迅速な対応が求められます。

【必須条件】
・プログラミング経験（Python, JavaScript他）がある方。
・Webサイトのスクレイピングに関する知識、または経験がある方。
・APIに関する知識、または開発経験がある方。
・コミュニケーションのレスポンスが遅くない方。

【歓迎条件】
・市営・
