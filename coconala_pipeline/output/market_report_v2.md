## 6. 我々の強みとの適合性分析

- ✅ **Claude Code自体が開発ツール** → 実際にコードを書いて納品できる

- ⚠️ デモ添付が難しい → コード品質やREADMEで差別化する必要

## 7. 戦略的提言

### 推奨戦略: 二刀流（Web制作 + IT/AIツール開発）

#### Tier 1: 最優先ターゲット

**A. IT/AI ツール開発（高単価・低競争）**
- 対象: Python自動化ツール、API連携、スクレイピング、Chrome拡張
- 予算帯: 3〜10万円（見積り希望案件は5〜15万で提案）
- 勝因: Claude Codeで高速開発 → 他の個人開発者より圧倒的に速い
- 差別化: GitHub上の動くコード + テスト済み成果物

**B. Web制作デモ付き提案（中単価・デモで差別化）**
- 対象: HP新規制作、LP、コーポレートサイト
- 予算帯: 3〜5万円
- 勝因: デモサイト + AI画像 → 提案時点で「完成品イメージ」を見せる
- 差別化: 他の提案者は口だけ、我々は実物を見せる

#### Tier 2: 積極的に狙う

**C. AI活用・導入相談**
- 対象: 画像生成AI活用、チャットAI活用、AIアプリ開発
- 予算帯: 見積り希望が多い → 10〜30万円で提案
- 勝因: 実際にAIを使いこなしている実績（Nano Banana Pro、Claude Code）
- 差別化: 「AIの専門家」としてブランディング

#### Tier 3: 余力があれば

**D. VBA/GAS/Excel自動化**
- 安定需要だが単価低め（5千〜1万円）
- Claude Codeで瞬殺できる案件が多い → 数をこなす戦略

### 狙うべき案件の条件

```
優先度スコア = (予算 × 0.4) + (低競争度 × 0.3) + (スキル適合度 × 0.3)

理想の案件:
  ✅ 見積り希望 or 3万円以上
  ✅ 提案数 10件以下
  ✅ Python / Web / AI / ツール系
  ✅ 投稿から3日以内（新鮮）

避けるべき案件:
  ❌ 提案30件以上
  ❌ 5千円未満（工数対効果が悪い）
  ❌ MQL/PLC/VSTなどニッチ専門技術
  ❌ 100万円以上（大型すぎて個人では厳しい）
```
