## 7. 戦略的提言

### 狙うべきゾーン

1. **「見積り希望」案件（36%）を優先ターゲットに**
   - クライアントが相場感を持っていない → 3〜5万円で提案してもリーズナブルに見える
   - デモサイトを添付することで「この品質でこの価格」の説得力

2. **投稿直後（提案5件以下）の案件を狙う**
   - 全体の18%しかないが、勝率が圧倒的に高い
   - パイプラインの自動化で「早期発見→即提案」が可能

3. **3万〜5万円ゾーンが最も効率的**
   - 件数が最多（21%）
   - 高予算帯ほどの激戦にならない
   - HP制作1件で十分な利益

### 差別化ポイント

1. **AI画像生成（Nano Banana Pro）** → 他の提案者にない圧倒的な差別化
2. **デモサイト即時提出** → 「口だけ」の提案者との差が明確
3. **レスポンシブ + アニメーション** → GSAP/ScrollTrigger対応は言及されにくいがインパクト大
4. **スピード** → 自動化パイプラインで提案速度が他を圧倒

### リスクと注意点

- 平均22件の提案競争 → 品質勝負になる
- 低予算案件（5千円未満）は工数対効果が悪い → スキップ推奨
- 69件提案の激戦案件 → 高予算でも勝率が低い。避けるか、デモ添付で差をつける
- Coconala手数料: 売上の22%が引かれる → 実手取りは提案額の78%
