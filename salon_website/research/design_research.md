## 1. Website Sections & Structure

A top-tier Japanese beauty salon website needs the following pages and sections. The structure follows the conventions established by successful Japanese salon websites, where customers expect specific information presented in a familiar hierarchy.

### 1.1 Primary Pages (Required)

| Page | Japanese Name | Purpose |
|------|--------------|---------|
| **Top / Hero** | トップページ | First impression, brand atmosphere, key visuals, CTA to book |
| **Concept** | コンセプト | Salon philosophy, brand story, what makes this salon unique |
| **Menu & Pricing** | メニュー・料金 | All services organized by category (Cut, Color, Perm, Treatment, Head Spa, etc.) with clear pricing |
| **Stylist Profiles** | スタイリスト紹介 | Individual profiles with photo, bio, specialties, years of experience, SNS links |
| **Style Gallery / Portfolio** | スタイルギャラリー | Hairstyle catalog organized by length, color, style, face shape. Before/after examples |
| **Salon Information** | 店舗情報 | Address, phone, hours, holidays, payment methods, parking, seat count |
| **Access / Map** | アクセス | Embedded Google Map, directions from nearest station(s), walking time, photos of the route |
| **Reservation** | ご予約 | Online booking form or link to HotPepper Beauty / external booking system |
| **First-Time Visitors Guide** | 初めての方へ | What to expect, salon flow, consultation process, FAQ for new customers |

### 1.2 Secondary Pages (Highly Recommended)

| Page | Japanese Name | Purpose |
|------|--------------|---------|
| **Blog / Column** | ブログ・コラム | Hair care tips, trend articles, product reviews, stylist diary entries |
| **Coupon / Campaign** | クーポン・キャンペーン | First-visit discounts, seasonal offers, referral programs |
| **News / What's New** | お知らせ | Holiday notices, new menu items, staff changes, event announcements |
| **Recruit / Careers** | 採用情報・リクルート | Job openings for stylists, assistants, receptionists |
| **Contact** | お問い合わせ | Contact form, LINE official account link, phone number |
| **Privacy Policy** | プライバシーポリシー | Required by law, especially for booking data handling |

### 1.3 Optional / Advanced Pages

| Page | Japanese Name | Purpose |
|------|--------------|---------|
| **Products / EC** | 取扱い商品 | Salon-exclusive products, hair care items with purchase links |
| **Hair Care Guide** | ヘアケアガイド | Educational content on hair care, color maintenance, seasonal tips |
| **Customer Reviews** | お客様の声 | Testimonials with photos (or links to HotPepper reviews) |

## 2. Design Trends 2025-2026

### 2.1 Overarching Aesthetic Directions

#### A. Airy / Light / Whitespace-Dominant (空気感重視)
The dominant trend in Japanese salon web design. Generous whitespace with a white or off-white base creates a clean, breathable feeling that mirrors the salon's atmosphere. Content floats in space rather than being boxed.

- White or warm off-white backgrounds (#FAFAF8, #F8F6F2)
- Large padding and margins between sections
- Asymmetric layouts that feel organic rather than rigid
- Thin typography, often in light or extra-light weights
- Minimal UI chrome; the design "disappears" behind the content

#### B. Editorial / Magazine Style (雑誌風)
Websites designed to feel like flipping through a high-end fashion magazine. Large typography as design elements, editorial photo layouts, and sophisticated grid-breaking compositions.

- Large hero typography overlapping images
- Multi-column editorial grids
- Full-bleed photography mixed with generous whitespace
- Feature article-style blog layouts
- Black and white as base with strategic color accents

#### C. Warm Natural / Botanical (ナチュラル・ボタニカル)
Popular with organic/natural-focused salons. Uses earthy tones, natural textures, and botanical motifs.

- Beige, cream, sage green, terracotta palettes
- Organic shapes and rounded corners
- Natural texture backgrounds (paper, linen, wood grain)
- Hand-drawn illustrations or botanical motifs
- Serif or rounded Gothic typography

#### D. Modern Minimal / Monochrome Chic (モダンミニマル)
For high-end, fashion-forward salons. Stark contrasts, bold typography, and a gallery-like presentation.

- Black/white/charcoal base
- Dramatic use of negative space
- Bold sans-serif typography as design element
- Grid-based portfolio layouts
- Subtle micro-animations on scroll

### 2.2 Specific Design Trends for 2025-2026

| Trend | Description | Application |
|-------|-------------|-------------|
| **Micro-animations** | Gentle parallax, breathing elements, cursor-reactive components. Movement supports clarity rather than overshadowing content. | Hero sections, scroll transitions, hover states on gallery images |
| **Scroll-driven storytelling** | Parallax scrolling creates layered depth for immersive browsing. Content unfolds as a narrative. | Salon concept sections, stylist introductions, before/after reveals |
| **Soft gradients & layered depth** | Fluid shapes with soft gradients replace flat backgrounds, adding subtle dimension. | Background elements, section dividers, card overlays |
| **Dynamic Instagram integration** | Live Instagram feeds embedded directly, creating constantly-updating galleries. | Gallery section, footer, sidebar showcasing latest work |
| **Video hero sections** | Short looping video clips showing the salon atmosphere, styling process, or brand mood. | Homepage hero, concept page |
| **Dark mode / night palette option** | Some progressive salons offer a dark mode toggle for a premium feel. | Site-wide toggle |
| **Cursor-following effects** | Subtle cursor interactions that create a playful, premium feel. | Gallery hover effects, navigation |
| **Split-screen layouts** | Image on one side, text on the other, creating visual balance. | Stylist profiles, concept sections |
| **Rounded UI elements** | Soft border-radius on buttons, cards, and images. Conveys approachability. | CTAs, cards, image treatments |
| **Sticky/floating booking CTA** | Ever-present reservation button that follows the user. | Fixed position button or bottom bar on mobile |

### 2.3 Things to Avoid

- Overly complex animations that slow page load
- Stock photos that feel generic (Japanese customers can spot them immediately)
- Cluttered layouts with too many competing elements
- Small, hard-to-tap booking buttons on mobile
- Auto-playing audio or intrusive popups
- Outdated design patterns (heavy drop shadows, beveled buttons, Flash-era aesthetics)

## 4. Typography: Japanese + English Font Pairings

### 4.1 Understanding Japanese Web Typography

Japanese web typography requires careful selection because:
- Japanese fonts are significantly larger in file size (thousands of characters vs. ~200 for Latin)
- Google Fonts serves Japanese fonts using `unicode-range` subsetting for performance
- Mixing Japanese and English fonts requires attention to baseline alignment and weight matching
- Japanese text does not use word spacing; line-height and letter-spacing need specific tuning

### 4.2 Recommended Font Pairings

#### Pairing A: Elegant Minimal (エレガントミニマル)
*Best for: High-end salons, editorial style, monochrome chic palette*

| Role | Japanese Font | English Font | Weight |
|------|--------------|-------------|--------|
| **Headings** | Shippori Mincho | Cormorant Garamond | 400-500 |
| **Subheadings** | Noto Sans JP | Josefin Sans | 300-400 |
| **Body** | Noto Sans JP | Josefin Sans | 300-400 |
| **Accent/Display** | Shippori Mincho B1 | Playfair Display | 400-700 |

- **Shippori Mincho**: An old-style Mincho (serif) with refined elegance. Bridges traditional Japanese aesthetics with modern readability. Available on Google Fonts.
- **Cormorant Garamond**: A refined Renaissance-inspired serif with high contrast. Excellent for display headings and fashion/editorial contexts.
- **Josefin Sans**: A vintage-geometric sans-serif with elegant thin weights. Works beautifully at light weights (300) for a high-fashion feel.
- **Playfair Display**: Bold editorial serif for accent text and feature headlines.

```css
/* Google Fonts import */
@import url('https://fonts.googleapis.com/css2?family=Shippori+Mincho:wght@400;500;600&family=Cormorant+Garamond:ital,wght@0,300;0,400;0,500;1,400&family=Josefin+Sans:wght@300;400;500&family=Noto+Sans+JP:wght@300;400;500&display=swap');
```

#### Pairing B: Soft & Approachable (ソフト・アプローチャブル)
*Best for: Casual/friendly salons, natural palette, soft pastel palette*

| Role | Japanese Font | English Font | Weight |
|------|--------------|-------------|--------|
| **Headings** | Zen Maru Gothic | Quicksand | 400-500 |
| **Subheadings** | Zen Maru Gothic | Quicksand | 300-400 |
| **Body** | Noto Sans JP | Quicksand | 300-400 |
| **Accent/Display** | Zen Maru Gothic | Josefin Sans | 300-400 |

- **Zen Maru Gothic**: A rounded Gothic (sans-serif) family that gives a soft, natural impression. The deep rounds in the corners create a friendly, approachable character. 5 weights from Light to Black. Available on Google Fonts.
- **Quicksand**: A geometric sans-serif with rounded terminals. Perfect companion for rounded Japanese fonts with its friendly, modern character.

```css
/* Google Fonts import */
@import url('https://fonts.googleapis.com/css2?family=Zen+Maru+Gothic:wght@300;400;500;700&family=Quicksand:wght@300;400;500;600&family=Noto+Sans+JP:wght@300;400;500&display=swap');
```

#### Pairing C: Clean Contemporary (クリーンコンテンポラリー)
*Best for: Modern salons, clean design, any palette*

| Role | Japanese Font | English Font | Weight |
|------|--------------|-------------|--------|
| **Headings** | Zen Kaku Gothic New | Montserrat | 300-500 |
| **Subheadings** | Noto Sans JP | Montserrat | 300-400 |
| **Body** | Noto Sans JP | Montserrat | 300-400 |
| **Accent/Display** | Zen Kaku Gothic New | Montserrat | 200-300 |

- **Zen Kaku Gothic New**: A simple Gothic font with a unique geometric style, offering a modern touch. Clean lines and balanced proportions ensure excellent readability. Available on Google Fonts.
- **Montserrat**: The workhorse of modern web design. Its geometric structure and extensive weight range (100-900) make it versatile. At thin weights (200-300), it creates an elegant, airy feel.
- **Noto Sans JP**: The "Helvetica of Japanese web fonts." Ubiquitous, highly legible, and reliable. Ensures consistent rendering across all devices.

```css
/* Google Fonts import */
@import url('https://fonts.googleapis.com/css2?family=Zen+Kaku+Gothic+New:wght@300;400;500;700&family=Montserrat:ital,wght@0,200;0,300;0,400;0,500;0,600;1,300;1,400&family=Noto+Sans+JP:wght@300;400;500&display=swap');
```

#### Pairing D: Luxe Serif (ラグジュアリーセリフ)
*Best for: Upscale salons, wedding/special occasion focus, editorial palette*

| Role | Japanese Font | English Font | Weight |
|------|--------------|-------------|--------|
| **Headings** | Noto Serif JP | EB Garamond | 400-500 |
| **Subheadings** | Noto Sans JP | Lato | 300-400 |
| **Body** | Noto Sans JP | Lato | 300-400 |
| **Accent/Display** | Shippori Mincho | EB Garamond | 400-600 |

- **Noto Serif JP**: A classic serif (Mincho) font by Google. Elegant and highly legible, suitable for both headings and body text. Adds refinement and a literary quality.
- **EB Garamond**: A faithful revival of Claude Garamond's original typefaces. Highly refined and suitable for luxury branding and editorial design.
- **Lato**: A humanist sans-serif that pairs beautifully with serif headings. Its warmth and clarity make body text highly readable.

```css
/* Google Fonts import */
@import url('https://fonts.googleapis.com/css2?family=Noto+Serif+JP:wght@400;500;600;700&family=EB+Garamond:ital,wght@0,400;0,500;0,600;1,400;1,500&family=Lato:ital,wght@0,300;0,400;1,300;1,400&family=Noto+Sans+JP:wght@300;400;500&display=swap');
```

### 4.3 Typography CSS Best Practices for Japanese Websites

```css
/* Base Japanese typography settings */
body {
  font-family: 'Noto Sans JP', 'Hiragino Kaku Gothic ProN', 'Hiragino Sans',
               Meiryo, sans-serif;
  font-size: 15px; /* 14-16px is standard for Japanese body text */
  line-height: 1.8; /* Japanese text needs more line-height than English (1.8-2.0) */
  letter-spacing: 0.05em; /* Slight letter-spacing improves readability */
  font-feature-settings: "palt"; /* Proportional alternates for balanced spacing */
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

h1, h2, h3 {
  line-height: 1.4; /* Tighter for headings */
  letter-spacing: 0.08em; /* Slightly more spread for headings */
}

/* English text within Japanese context */
.en-text {
  letter-spacing: 0.02em;
  line-height: 1.6;
}
```

## 5. Must-Have Features

### 5.1 Online Booking System (予約システム)

#### HotPepper Beauty Integration
HotPepper Beauty (ホットペッパービューティー) is Japan's largest salon booking platform with over 100,000 registered salons. Integration is essential.

**Implementation approaches:**

1. **Direct Link (Most Common)**: Prominent "HotPepper Beautyで予約" button linking to the salon's HotPepper page. Simple, reliable, and maintains the platform's booking flow.

2. **HotPepper Beauty Widget**: Embed a booking widget directly on the salon website for a seamless experience.

3. **Unified Booking Management (BMS)**: Use platforms like CYND's Booking Manager that integrate HotPepper Beauty alongside other booking channels, syncing availability in real-time across 11+ booking sites.

4. **Proprietary Booking Form**: Custom booking form on the salon website that feeds into salon management software (e.g., Salon Answer, SALONPOS, Beauty Merit).

**Booking CTA best practices:**
- Booking button must be visible within 2 seconds of page load (60%+ of potential clients leave if it takes longer to find)
- Use a sticky/floating "Reserve" button on mobile
- Include booking CTAs in the header, after each service description, and in the footer
- Show phone number as a tap-to-call link on mobile for customers who prefer phone booking

#### LINE Official Account Booking
Many Japanese salons now accept reservations via LINE. Include a LINE friend-add button prominently.

### 5.2 Stylist Portfolio System (スタイリストポートフォリオ)

Each stylist should have:
- Professional portrait photo
- Self-introduction / greeting message (自己紹介)
- Years of experience and career history
- Specialty techniques (得意な技術): e.g., balayage, bob cuts, scalp treatment
- Style gallery linked to their work
- Personal SNS links (Instagram is most important)
- Direct booking link filtered to that stylist
- Message / comment from the stylist (a personal touch)

### 5.3 Price Menu (料金メニュー)

**Structure recommendations:**
- Organize by service category (Cut / Color / Perm / Treatment / Head Spa / Other)
- Show prices in tax-included format (tax-included pricing is legally required in Japan since April 2021)
- Indicate estimated time for each service
- Highlight popular/recommended menus
- Include set menus and combination discounts
- Separate pricing for first-time visitors vs. regular customers when applicable
- Note any rank-based pricing differences (e.g., stylist vs. top stylist vs. director pricing)

### 5.4 Coupon / First-Visit Offers (クーポン・初回限定)

- Dedicated first-visit coupon page or section
- Clear conditions (valid period, applicable services, restrictions)
- Eye-catching design that stands out from regular content
- QR codes or coupon codes that can be shown at the salon
- Seasonal campaign promotions
- Referral program information (friend referral discounts)

### 5.5 SNS Integration (SNS連携)

**Priority platforms for Japanese beauty salons:**

1. **Instagram** (Highest priority): Live feed embed, story highlights, link to profile. The primary portfolio platform for Japanese stylists.
2. **LINE Official Account**: Friend-add button, chat booking, push notifications for campaigns.
3. **TikTok**: Increasingly important for reaching younger demographics. Embed popular videos.
4. **X (formerly Twitter)**: Real-time updates, salon news, personality-driven content.
5. **Google Business Profile**: Not a traditional SNS, but critical for local SEO and maps visibility.

**Integration implementation:**
- Instagram feed widget in gallery section (auto-updating)
- SNS icon links in header and/or footer
- LINE friend-add popup or floating button
- Share buttons on blog posts and style gallery items
- Embedded video content from TikTok/YouTube

### 5.6 Blog / Column (ブログ・コラム)

**Content strategy for salon blogs:**
- Hair care tips and seasonal advice (季節のヘアケア)
- Trend reports and new styles (トレンド情報)
- Product introductions and reviews (商品紹介)
- Stylist diary / behind-the-scenes (スタイリスト日記)
- Before/after transformations with explanations
- FAQ-style articles answering common customer questions
- Salon news and event reports

**SEO benefits:** Blog content helps with long-tail keyword ranking and drives organic traffic from search engines.

## 6. Image & Asset Needs

### 6.1 Required Photography Categories

| Category | Quantity (Min.) | Specifications | Usage |
|----------|----------------|----------------|-------|
| **Hero / Key Visuals** | 3-5 | Full-width, high resolution (2400px+ wide), landscape orientation. Atmosphere shots that capture the salon's mood. | Homepage hero slider/banner, concept page |
| **Hairstyle Portfolio** | 30-50+ | Consistent lighting and background. Front, side, and back angles. Tagged by length, color, and style. | Style gallery, stylist portfolios |
| **Before & After** | 10-20 | Side-by-side or slider format. Same lighting conditions. Close attention to consistent framing. | Gallery, blog, social media |
| **Stylist Portraits** | 1-2 per stylist | Professional headshot + casual/working shot. Warm, approachable expressions. Consistent style across all stylists. | Stylist page, about page |
| **Salon Interior** | 8-12 | Reception, styling stations, shampoo area, waiting area, products display. Both wide and detail shots. Shot during golden hour or after sunset for best lighting. | Salon info, concept, first-visit guide |
| **Exterior / Entrance** | 2-3 | Building exterior, entrance, signage. Help customers recognize the salon when they arrive. | Access page, first-visit guide |
| **Treatment Process** | 5-8 | Hands-on styling, coloring, shampooing, head spa, consultation scenes. Conveys care and expertise. | Concept page, menu page, blog |
| **Product Shots** | 3-5 | Salon-used or salon-sold products. Clean product photography on neutral background. | Product page, blog |
| **Detail / Atmosphere** | 5-10 | Close-ups of tools, flowers, interior details, textures. These fill gaps and add sophistication. | Background elements, blog headers, section decoration |

### 6.2 Photography Guidelines

**Technical Requirements:**
- Minimum resolution: 2400 x 1600px for hero images, 1200 x 800px for gallery
- File format: JPEG for photos, WebP for web delivery (with JPEG fallback)
- Color profile: sRGB for web consistency
- File size: Optimize to under 200KB for gallery thumbnails, under 500KB for full-size

**Aesthetic Guidelines:**
- Natural or warm lighting preferred (avoid harsh flash)
- Consistent color grading across all photos
- Clean, uncluttered backgrounds
- Real customers or models (avoid obvious stock photos; Japanese customers are discerning)
- Diversity of hair lengths, colors, and styles represented
- Best time for interior shots: after sunset, when artificial lighting design is most flattering

**Before/After Photography Protocol:**
- Same distance, angle, and lighting for both shots
- Neutral background (white or light gray)
- Hair natural (not styled with product) in "before" shot
- Include face (with permission) or shoot from behind for privacy
- Note: Always obtain written consent (撮影同意書) from models/customers

### 6.3 Video Assets (Optional but Recommended)

| Type | Duration | Usage |
|------|----------|-------|
| Hero background loop | 10-15 sec | Homepage hero section |
| Salon tour | 30-60 sec | Concept or salon info page |
| Styling process timelapse | 15-30 sec | Gallery, social media |
| Stylist introduction clips | 15-20 sec each | Stylist profile pages |

## 7. UX Best Practices

### 7.1 Mobile-First Design (Critical)

**Why mobile-first is non-negotiable:**
- 90%+ of beauty salon website traffic in Japan comes from smartphones
- Most bookings happen on mobile, often from search results or SNS links
- HotPepper Beauty's mobile app is the primary booking channel
- Google uses mobile-first indexing for search rankings

**Mobile-first implementation:**
- Design for 375px width first (iPhone SE/standard), then scale up
- Touch targets minimum 48x48px with adequate spacing to prevent mis-taps
- Hamburger menu or sticky bottom navigation bar
- Sticky bottom CTA bar with "Reserve" and "Call" buttons
- Swipeable image galleries and carousels
- Collapsible/accordion menus for pricing tables
- Avoid hover-dependent interactions (no hover on mobile)
- Lazy-load images below the fold

### 7.2 Booking Conversion Optimization

**Friction reduction:**
1. The booking button should be findable within 2 seconds of landing
2. Limit booking form fields to essentials (name, phone/email, date, time, service, stylist preference)
3. Show real-time availability when possible
4. Provide multiple booking channels: web form, HotPepper, LINE, phone
5. Send immediate confirmation via email and/or LINE
6. Display clear cancellation policy

**Trust-building elements:**
- Customer review scores (link to HotPepper reviews)
- Number of years in business / number of customers served
- Staff qualification badges and certifications
- Clear pricing with no hidden fees
- Before/after transformations as social proof
- Google Maps embed with reviews

**Conversion-focused layout:**
- "Reserve" CTA button should use the accent color and be the most visually prominent element
- Place booking CTAs after service descriptions, stylist profiles, and gallery sections
- Use urgency indicators: "Today's available slots" or "Popular time slots filling up"
- First-visit coupon should be visible on the first screen (above the fold)

### 7.3 Performance Optimization

| Metric | Target | Why |
|--------|--------|-----|
| **First Contentful Paint (FCP)** | Under 1.5 seconds | Users perceive the page as loading fast |
| **Largest Contentful Paint (LCP)** | Under 2.5 seconds | Core Web Vital; Google ranking factor |
| **Cumulative Layout Shift (CLS)** | Under 0.1 | Prevents annoying content jumping |
| **Total page weight** | Under 3MB | Fast load on mobile networks |
| **Time to Interactive (TTI)** | Under 3 seconds | 32% bounce rate increase from 1s to 3s load time |

**Optimization techniques:**
- Use WebP/AVIF image formats with JPEG fallback
- Implement responsive images with `srcset` and `sizes`
- Lazy-load images below the fold (use `loading="lazy"`)
- Subset Japanese fonts using `unicode-range` (Google Fonts does this automatically)
- Preload critical fonts and hero images
- Use CDN for static assets
- Enable Brotli/Gzip compression
- Minimize third-party scripts (especially social media embeds; load on scroll or click)

### 7.4 Visual Portfolio Optimization

**Gallery UX patterns:**
- Masonry or grid layout with consistent aspect ratios
- Filter/sort by: hair length, color, style type, stylist, face shape
- Lightbox view with swipe navigation
- Each style entry should link to the stylist who created it
- Include a "Book this style" CTA on each gallery item
- Load more / infinite scroll rather than pagination (for mobile)

**Image loading strategy:**
- Show low-resolution placeholder (blur-up technique) while loading
- Load thumbnail grid first, full resolution on tap/click
- Preload next few images in lightbox view
- Use `aspect-ratio` CSS to prevent layout shift

### 7.5 Accessibility & Inclusivity

- Sufficient color contrast (WCAG AA minimum: 4.5:1 for normal text)
- Alt text on all images (describe the hairstyle for screen readers)
- Keyboard navigable booking form
- Font size minimum 14px for Japanese body text
- Resizable text without layout breaking
- Focus indicators on interactive elements

### 7.6 SEO for Beauty Salons

**Critical SEO elements:**
- Title tag format: `[Salon Name] | [Area Name] Beauty Salon` (e.g., "Paxini | 渋谷 美容室")
- Meta descriptions with location, key services, and USP
- Structured data (Schema.org): `HairSalon`, `LocalBusiness`, `Service`
- Google Business Profile optimization with photos, hours, and services
- Blog content targeting long-tail keywords (e.g., "渋谷 髪質改善 トリートメント")
- Internal linking between style gallery, stylist profiles, and service pages
- Mobile-friendly test passing
- Page speed optimization (Core Web Vitals)
